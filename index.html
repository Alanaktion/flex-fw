A new grid system

Flexbox makes it easy to build a powerful grid system, with support for all kinds of fun things that previously required crazy hacks or simply weren't possible.

These examples display best on a larger screen than what you're using now, we recommend getting a tablet or laptop to see what's going on.

Basic grid

Each row can include traditional percent-based columns as well as dymamically-sized col-*-auto columns. We've used the Bootstrap 4 grid class conventions and breakpoints to add the familiar container, row, and col-xs-* through col-xl-* classes. Just like in Bootstrap, they can be modified to any level desired including adding more breakpoints and changing the viewport sizes through simple modifications to the _variables.scss file.
